# tree-generator

这是一个可以根据指定目目录生成目录树的插件，形如

```js
  ├─ css              // css
  │  ├─ common.css    // common.css
  │  └─ details.css   
  ├─ images           
  │  └─ logo.png      //
  └─ js               
     ├─ lib           
     │  └─ jquery.js  //
     ├─ common.js     
     ├─ home.js       
     └─ pages.js      //

```

## Features

* 美观。连接线完美展示，注释位对齐
* 方便。只需右键点击生成即可复制到剪贴板上
* 快速。原生实现，不依赖任何第三方，性能高

> 已排除已`.`开头的文件或文件夹和`node_modules`文件夹
